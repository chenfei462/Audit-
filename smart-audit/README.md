# AuditLens — 会计智能查账系统 v0.2

> 基于 Rust 的高性能财务审计工具，支持命令行和 Web 双模式

## 功能特性

### 数据导入
- CSV / Excel (.xlsx/.xls) 文件导入
- MySQL / PostgreSQL 数据库直连

### 审计规则（8 条）
| 规则 | 说明 |
|------|------|
| 借贷平衡检查 | 校验每笔凭证借贷是否相等 |
| 金额阈值检测 | 标记大额交易 |
| 重复交易识别 | 检测日期+金额+科目的重复 |
| 日期连续性检查 | 检测日期间断和时序错误 |
| 整数金额预警 | 频繁整数金额可能为虚构交易 |
| Benford 定律 | 首位数字分布是否异常 |
| 关联交易-资金回流 | 同日同金额多次出现 |
| 关联交易-期末集中 | 月末集中大额交易 |

### 异常检测
- Z-Score 统计检测，支持按科目分组

### 输出方式
- 终端彩色报告
- JSON / CSV 导出
- **Web 可视化界面**（含 4 种图表）

## 快速开始

```bash
# 编译
cargo build --release

# 命令行审计
cargo run -- audit -i data.csv -f csv

# 启动 Web 界面
cargo run -- web --port 8080
# 然后浏览器打开 http://127.0.0.1:8080
```

## 测试

```bash
cargo test
```
